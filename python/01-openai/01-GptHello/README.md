## 用法

1. 登入 OpenAI
2. 選右上角你帳戶功能表中的 View API key
3. 按 Create a new secret key
4. 把 key 設到 OPENAI_API_KEY 這個環境變數中
5. 執行 GptHello.py 程式

## 執行過程

```
$ pip install openai
...
$ echo $OPENAI_API_KEY
...
```
